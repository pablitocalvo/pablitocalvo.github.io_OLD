installare eclipse su tiniX

  • scaricare eclipse package da qui : eclipse page (198 MiBi)
  • aprire un terminale e digitare :

    aprire un terminale: e digitare il comando:
    user@tiniX:~$ sudo su
    il terminale dovrebbe apparire ora nel seguente modo:
    root@tiniX:/home/user#
  • digitare il comando :
    root@tiniX:/home/user# cd /opt/
    il terminale dovrebbe apparire ora nel seguente modo:
    root@tiniX:/opt#
  • scompattare il file scaricato con il comando : root@tiniX:/opt# tar xvfz /home/user/Scaricati/eclipse-java-2019-09-R-linux-gtk-x86_64.tar.gz
  • uscire dalla utenza root: root@tiniX:/opt# cd /home/user; exit
    ora il terminale dovrebbe apparire cosi: user@tiniX:~$
  • creare icona sul desktop ...
    user@tiniX:~$ pluma Scrivania/Eclipse.desktop
    copiare e incollare le seguenti righe: #!/usr/bin/env xdg-open
    [Desktop Entry]
    Name=Eclipse Java IDE
    Comment=Eclipse Java IDE
    Type=Application
    Encoding=UTF-8
    Exec=/opt/eclipse/eclipse
    Icon=/opt/eclipse/icon.xpm
    Categories=GNOME;Application;Development;
    Terminal=false
    StartupNotify=true
    Sakvare il file e bla bla bla ....
